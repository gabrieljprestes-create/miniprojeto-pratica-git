# Miniprojeto: Gerador de Cores Aleatórias

Este projeto foi desenvolvido como parte de uma atividade prática sobre Git e GitHub.

## Objetivo
Aplicar conceitos de versionamento, branches, commits, issues e pull requests utilizando um miniprojeto simples em HTML, CSS e JavaScript.

## Funcionalidades
- Geração de cores hexadecimais aleatórias.
- Alteração dinâmica da cor de fundo da página.
- Exibição do código da cor gerada.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (ES6+)
- Git & GitHub
